Hermes_Project
=============
real-time interpretation system for the deaf
----

**1. 프로젝트 개요**

>손 모양으로 한글을 표현한 '지문자'를 이용하여 청각장애인과 건청인이 각자의 편한 방식으로 대화할 수 있도록 통역하는
>실시간 통역 및 생활 소리 알림 시스템
